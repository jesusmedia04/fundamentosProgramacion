Condicionales en JavaScript

Son estructuras que permiten tomar decisiones basadas en condiciones verdaderas o falsas.

Se utilizan las palabras clave if, else if, y else.

Ejemplo:


        if (condición) {
          // Código si la condición es verdadera
        } else {
          // Código si la condición es falsa
        }
Ciclos en JavaScript

Son estructuras que permiten ejecutar repetidamente un bloque de código mientras se cumple una condición.

Los ciclos más comunes son for y while.

Ejemplo de un ciclo for:


        for (let i = 0; i < 5; i++) {
          // Código a ejecutar en cada iteración
        }

Ejemplo de un ciclo while:


        let i = 0;
        while (i < 5) {
          // Código a ejecutar en cada iteración
          i++;
        }
Algoritmos

Son conjuntos finitos de pasos o instrucciones que describen un proceso para resolver un problema o realizar una tarea específica.

Los algoritmos son la base de la programación y pueden ser representados de manera abstracta o en forma de diagramas.

Ejemplo:


        // Algoritmo para sumar dos números
        1. Ingresar dos números A y B.
        2. Sumar A y B y almacenar el resultado en C.
        3. Mostrar el valor de C.
